Proyectos

CRONÓMETRO APP

Academlo, 2020

Creado con el framework React nos permite visualizar y manejar un cronómetro. Para iniciarlo se da clic en el botón de start, se puede pausar el tiempo con el botón de pausa y si se desea también se puede imprimir el tiempo desde que se dio click a start y por último se puede resetear el cronometro poniendo el tiempo en cero. https://github.com/glenda123/cronometro_glenda.git

TODO APP

Academlo 2020

Creado con el framework React que nos permite agregar tareas, listar las tareas, eliminar las tareas y tachar las tareas en el caso que las tareas ya hayan sido realizadas. Las tareas se quedan guardadas en el localstorage. https://github.com/glenda123/todo_list_glenda.git

HÉROES APP

Academlo, 2020

Creado con React consta de una sencilla página compuesta de un navbar y el contenido principal. El navbar consta de tres componentes, el componente Marvel Screen que me permite visualizar a los super héroes de Marvel, el segundo componente llamado DC Screen me permite visualizar también super héroes pero que no pertenecen a Marvel, el componente Search me permite buscar a los super héroes por su nombre y por último el componente Logout que al dar click nos saca de la aplicación. Al cargar la aplicación se muestra solamente el boton de login y con sólo dar click ya nos permite entrar en la aplicación. https://github.com/glenda123/heroes_app_glenda.git
